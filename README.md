# Banking Sales BI Dashboard

## Overview
This project is a beginner-friendly Business Intelligence dashboard created to analyse banking product sales and regional performance.

## Tools Used
- Power BI
- Excel
- CSV Dataset

## Key Features
- Total sales overview
- Sales by banking product
- Regional sales analysis
- Time-based sales trends

## Insights
- Home Loans generated the highest sales
- Gauteng showed strong regional performance
- Sales trends increased steadily over time

## Dataset
The dataset used in this project is simulated and created for educational purposes.
